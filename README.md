An efficient, lightweight, and serverless examination platform designed specifically for college professors to seamlessly conduct, manage, and evaluate student assessments. 

## 🌐 Live Project
- **Production URL:** [msc.aiits.workers.dev](https://workers.dev)
- **Deployment Platform:** Cloudflare Workers & Render

---

## 📌 The Problem
Traditional online examination portals are often expensive, heavy, and require complex server management. For individual school or college professors, setting up an infrastructure to manage multiple student tests simultaneously leads to:
1. High computational costs and server downtimes during peak hours.
2. Complicated manual grading and feedback systems.
3. Lack of optimization for lower-end devices or slow internet speeds used by students.

## 🚀 The Solution
**AIITS-VIB** solves this by leveraging modern serverless computing and AI assistance. By offloading computational tasks to the edge using **Cloudflare Workers**, the application achieves near-zero latency, exceptional scalability, and operates at **zero hosting cost** for the educator. 

### Key Features:
- **Serverless Edge Architecture:** Deployed on Cloudflare Workers for global scalability and sub-millisecond response times.
- **AI-Assisted Workflows:** Developed using modern LLM capabilities (Claude AI) to design optimized data flows.
- **Robust Backend Logic:** Express.js and Node.js backend handles secure routing and test tracking efficiently.
- **Student-First UX:** Ultra-lightweight HTML/JS frontend ensuring the website loads effortlessly even on 3G connections.
