How Advertisements Work

1. Upload images here. They appear in the student home page gallery and student gallery page automatically if "Show in Home Page Gallery" is on.

2. To show an ad before a specific test: go to Main Admin → Create/Edit Test → enable "Ad Page" → select which images to show.

3. When a student clicks "Attempt" on that test, they see the selected images as a slideshow (5 seconds each) before the test timer starts.

4. Clicking an image opens the Click URL you set. Use this for coaching updates, announcements, sponsor links etc.
